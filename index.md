HTML Games
================

### Dependencies
You need node and gulp to run this project.


### Build Process

To build the project run the following:
* *npm install*
* *npm install -g gulp*


### Running the Project

Run the project by typing `gulp` in your 

The url of the app will be *http://localhost:3000*
